## Rules

You can use --include.
You can use --exclude.
You can use * for any char.
You can use ? for a single char.
You can use [...] for a "class."
You can use [!...] to negate a "class."

https://docs.aws.amazon.com/cli/latest/reference/s3/#use-of-exclude-and-include-filters
